# Tofaş Radyo

A free static Turkish radio player styled like an old in-car radio/cassette unit.

## What it does

- Loads Turkish radio stations from Radio Browser
- Uses only free public resources
- Plays stations in the browser
- Old car-radio inspired interface
- Search
- Preset-style filters
- Favorites
- Recently played
- Light/dark dashboard mode
- PWA support
- Runs on GitHub Pages
- No backend
- No login
- No paid services

## Project structure

```text
tofas-radyo/

  service-worker.js
```

## Deploy on GitHub Pages

1. Create a public GitHub repository called `tofas-radyo`.
2. Upload all files to the root of the repository.
3. Go to **Settings**.
4. Go to **Pages**.
5. Select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.

Your app will be available at:

```text
https://yourusername.github.io/tofas-radyo/
```

## Notes

This is an aesthetic tribute to old Turkish car radio/cassette dashboards. It is not affiliated with Tofaş or any car/radio brand.

Some stations may not play because public radio streams can be broken, blocked, or changed.
